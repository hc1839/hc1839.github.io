Determines the end (exclusive) index of the specified group in an interval with a known index of the first element in the group.

An interval encompasses a contiguous sequence of elements to be subdivided into groups.

It is meant to be called iteratively starting from the first group with index 0.

Parameters
interval_start_indexIndex of the first element in the interval.
interval_end_indexIndex of the element following the last element in the interval. It must be no less than interval_start_index.
num_groupsNumber of groups. It must be positive.
group_indexIndex of the group to determine the end index. First group in the interval has the index 0.
group_start_indexIndex of the first element in the group specified by group_index.
Returns
Index of the element following the last element in the group specified by group_index.
