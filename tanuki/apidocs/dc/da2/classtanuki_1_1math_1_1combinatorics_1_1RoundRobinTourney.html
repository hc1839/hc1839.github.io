Detailed Description

template<typename T>
class tanuki::math::combinatorics::RoundRobinTourney< T >

Creates rounds of a round-robin tournament as rows of a two-column matrix.

In a set of \( n \) competitors, each competitor is assigned an integer in the range \( [0, n) \). The circle method is used to create the rounds. For each round, a matrix is constructed such that each row represents pair of competitor indices. Iteration over the rounds are deterministic in order.

If there is an odd number of competitors, an index of \( -1 \) is added to the set of competitors as a dummy competitor. For any round, a competitor is idle when paired with \( -1 \).
