map< Spin, vector< void * > > tanuki::interop::SpinMap ( const CMultiArray &  marr,
bool  is_col_major,
bool  is_col_spin 
)

Gets the items in a two-dimensional multi-array as a spin map of opaque pointers to the items.

Parameters
marrTwo-dimensional multi-array.
is_col_majorWhether marr is in column-major order.
is_col_spinWhether spin corresponds by column in marr. If true, marr must have one or two columns. One column indicates no spin polarization, and two columns indicate spin polarization. For spin polarization, first column must correspond to Spin::UP, and second column must correspond to Spin::DOWN. If false, rows (instead of columns) are concerned.
Returns
Spin map of opaque pointers to the items in marr.
