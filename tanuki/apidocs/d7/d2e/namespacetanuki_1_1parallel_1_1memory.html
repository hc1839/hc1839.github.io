Copies bytes with OpenMP and MPI parallelization.

Parameters
mpi_commMPI communicator for further parallization with MPI, or MPI_COMM_NULL for OpenMP only. If not MPI_COMM_NULL, behavior is defined only when src points to a memory location (not necessarily shared) that has the same data contents across the processes and dst points to shared memory.
srcPointer to the memory location to copy from. It is reinterpreted as an array of char.
dstPointer to the memory location to copy to. It is reinterpreted as an array of char.
