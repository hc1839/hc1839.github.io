Whether two numbers are approximately equal within an absolute tolerance.

Template Parameters
TType of number. Supported types are tanuki::number::real_t and tanuki::number::complex_t. Other types are undefined.
Parameters
aFirst number.
bSecond number.
tolAbsolute tolerance.
Returns
For real numbers, returns true if \( \left| a - b \right| \leq \mathrm{tol} \). For complex numbers, corresponding real and imaginary parts are compared as for real numbers, and returns true if the corresponding real and imaginary parts are both approximately equal.

Whether two numbers are approximately equal within a relative tolerance.

Template Parameters
TType of number. Supported types are tanuki::number::real_t and tanuki::number::complex_t. Other types are undefined.
Parameters
aFirst number.
bSecond number.
tolRelative tolerance.
Returns
For real numbers, returns true if \( \left| a - b \right| / \max(\left| a \right|, \left| b \right|) \leq \mathrm{tol} \). For complex numbers, corresponding real and imaginary parts are compared as for real numbers, and returns true if the corresponding real and imaginary parts are both approximately equal.
