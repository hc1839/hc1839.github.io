Detailed Description

template<typename T>
class tanuki::state::SpinRangeTransformer< T >

Eager transformer of a range to spin ranges for effective spin enumerators.

For the definition of an effective spin enumerator, see SpinRange.

Template Parameters
TType of elements in a spin range.

FTransformation function.
Parameters
firstBeginning of the range to transform.
lastPast-the-end of the range to transform.
funTransformation function that is eagerly applied to the elements in [first, last). It must return a const T * given an element in [first, last) and a spin enumerator, respectively. The spin enumerator that is passed to the transformation function is exact and not effective. The transformation function must meet the requirement where it returns a non-nullptr given Spin::DUAL if and only if it returns nullptr given other spin enumerators.
