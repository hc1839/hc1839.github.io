Common.Float.nearlyEqual(a, b, epsilon=2.2250738585072014e-308)¶

Determines whether two floating-point numbers are considered to be nearly equal.

Adapted from “The Floating-Point Guide”.

Parameters:

  • a (float) – Floating-point number.

  • b (float) – Floating-point number.

  • epsilon (float) – Arbitrarily small positive number for testing difference.

Returns:

True if the difference between a and b is sufficiently small to be considered as nearly equal; False if otherwise.

Return type:

bool
