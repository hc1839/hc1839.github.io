class Special.FiniteElement.Element(mesh, nodes, shapeFns)¶

Bases: ABC

Finite element.

Subclasses must have constructors with the same required arguments as __init__().

Data members:

_mesh: See mesh().

_nodes: List of nodes. See nodes().

_shapeFns: List of local shape functions. See __init__().

_coordTransformation: See coordTransformation().

__init__(mesh, nodes, shapeFns)¶
Parameters:

  • mesh (Mesh) – Mesh that the element belongs to.

  • nodes (iterable) – Sequence of Node objects. Order is not important. Local indices are determined by element type and node positions.

  • shapeFns (iterable) – Sequence of shape functions, as SympyFn.SympyFn objects, ordered by local index. The shape functions are in parent element coordinates.

Raises:

  • ValueError – Number of nodes is zero.

  • ValueError – Number of shape functions is zero.

  • ValueError – Number of shape functions is not the same as number of nodes.

  • ValueError – Nodes do not have the same dimensionality.

  • ValueError – Shape functions do not have the same parameters.

  • ValueError – Shape functions do not have the same dimensionality as the nodes.

build(globalNodeIndexer)¶

Constructs a Mesh object from the scaffold.

Sanity checks should be performed.

Parameters:

globalNodeIndexer (callable) – Ternary function, where first parameter is new Mesh object that is being constructed, second parameter is sequence of nodes currently in the new mesh, and third parameter is newly constructed Node object that is not a placeholder and lacks a global index. It returns an int that specifies the global index of the node.

Returns:

Instance of a subclass of Mesh.

Return type:

subclass of Mesh

Raises:

  • ValueError – Two node placeholders of different types exist at the same position.
