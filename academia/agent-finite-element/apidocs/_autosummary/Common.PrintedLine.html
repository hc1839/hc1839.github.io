Common.PrintedLine¶

Common.PrintedLine.printWithCr

Prints a string with a carriage return.

Facilitates overwriting the printed line.

Common.PrintedLine.printWithCr(string, stream=<_io.TextIOWrapper name='<stdout>' mode='w' encoding='utf-8'>)¶

Prints a string with a carriage return.

It remembers the length of the previous string it printed and pads the string to be printed with spaces, if necessary, to overwrite the previously printed string.

Parameters:

  • string (str) – String to print.

  • stream – Stream to print to.

Returns:

None
