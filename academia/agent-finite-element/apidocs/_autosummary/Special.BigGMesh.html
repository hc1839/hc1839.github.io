Special.BigGMesh._meshFromNodePosIen(nodes, ien, isDirichletNode)¶

Constructs a mesh using right triangles.

Parameters:

  • nodes (numpy.ndarray) – Matrix of node positions as row vectors.

  • ien (numpy.ndarray) – IEN matrix of global node indices, where each row vector represents a triangle, and column vectors represent the local node indices in order.

  • isDirichletNode (callable) – Unary function, where first parameter is node position as a row vector. It returns True if the position corresponds to a Dirichlet node; False if otherwise.

Returns:

Mesh with isosceles right triangles as elements.

Return type:

FiniteElementTriangle.TriangleMesh

Raises:

  • ValueError – Global indices are not unique.

  • ValueError – Triangle is neither a lower nor an upper.
