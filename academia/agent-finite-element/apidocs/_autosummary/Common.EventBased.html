attach(observer, handler, *argv)¶

Attaches a function as an event handler.

Parameters:

  • observer (object) – Observer of the event.

  • handler (callable) – Function to be attached as a handler of the event. It is a member function of the class of observer and, therefore, must receive observer as its first argument.

  • argv – Any additional values.

Returns:

None

Raises:

ValueError – Observer is already listening to the event.

detach(observer)¶

Detaches an observer.

Parameters:

observer – Observer to be detached from receiving the event.

Returns:

None

Raises:

ValueError – Observer is not listening to the event.
