AtomIsland

class AtomIsland<A : Atom>(atom: A) : AbstractFragment<A> , Island<A>

Island representing a single atom.

It should be used only by an instance of Atom to construct an island no more than once such that there is one-to-one correspondence with the atom that the island is representing.
