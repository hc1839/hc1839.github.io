Production

enum Production : Enum<Production> , Reducer<Production> , ColorFiller<Production>

Production rules that are used as node types in a parse tree for the UCUM unit syntax.

Annotations are not supported. Therefore, curly brackets are not considered to be valid terminal symbols.

Except some additional productions, the meaning of the enum constant names follow the BNF as specified in the UCUM and are not documented here. Additional productions, on the other hand, are documented here.
