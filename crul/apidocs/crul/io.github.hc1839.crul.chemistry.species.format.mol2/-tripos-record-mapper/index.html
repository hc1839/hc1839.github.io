TriposRecordMapper

interface TriposRecordMapper

Mapper for transforming supported Tripos records when exporting a supermolecule to Mol2.

Supported Tripos record types are specified in TriposRecordMapper.supportedTypes. Tripos ATOM is not part of the mapper, since a supermolecule being exported must already have TriposAtom as the type of its atoms.
