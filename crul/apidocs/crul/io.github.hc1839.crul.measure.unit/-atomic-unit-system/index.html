AtomicUnitSystem

object AtomicUnitSystem : UnitSystem

Hartree atomic unit system.

The base unit for amount of substance is set to be the same as the corresponding SI unit, which is the mole.

Luminous intensity does not have an apparent atomic unit, so it is derived by first determining the irradiance (according to radiometry) when the electric field is set to 1 when in atomic units. The irradiance formula used is

irradiance = 0.5 * speedOfLight * permittivity * electricField^2

Then, a sphere with radius of 1 bohr is used to determine the radiant intensity from irradiance. Finally, the relative magnitude of the radiant intensity in atomic units with respect to 1 candela's radiant intensity of 1 / 683 W/sr defines the atomic unit of luminous intensity, which is approximately 671.215 cd.
